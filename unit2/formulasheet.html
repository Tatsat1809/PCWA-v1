Unit 2: Transformations & Mapping

Formula Sheet

Master Form & Parameter Roles

  • Master form: y = a \cdot f\big(b(x + c)\big) + d
  • a: vertical stretch/compression by |a|; reflect in x-axis if a < 0
  • b: horizontal compression/stretch; scale factor = 1/|b|; reflect in y-axis if b < 0
  • c: horizontal shift; +c moves left c units, −c moves right c units
  • d: vertical shift; +d up, −d down
  • Mapping rule: If (x, y) lies on y = f(x), then on y = a·f(b(x + c)) + d the corresponding point is:
    • x' = \frac{x}{b} - c
    • y' = a \cdot y + d
  • Order of effects: apply “inside” (b, c) first (horizontal), then “outside” (a, d) (vertical).

Quick Mapping: What Each Transformation Does

  • Vertical stretch by k (>1): y = k f(x) \rightarrow (x, k\cdot y)
  • Vertical compression by k (0<k<1): y = k f(x) \rightarrow (x, k\cdot y)
  • Vertical reflection: y = -f(x) \rightarrow (x, -y)
  • Horizontal compression by k (>1): y = f(kx) \rightarrow (\frac{x}{k}, y)
  • Horizontal stretch by k (>1): y = f(x/k) \rightarrow (k\cdot x, y)
  • Horizontal reflection: y = f(-x) \rightarrow (-x, y)
  • Horizontal shift right by c: y = f(x - c) \rightarrow (x + c, y)
  • Horizontal shift left by c: y = f(x + c) \rightarrow (x - c, y)
  • Vertical shift up by d: y = f(x) + d \rightarrow (x, y + d)
  • Vertical shift down by d: y = f(x) - d \rightarrow (x, y - d)

Domain, Range, Intercepts, Asymptotes

  • Domain: apply the inverse horizontal map to parent domain endpoints: x' = \frac{x}{b} - c
  • Range: apply vertical map to parent range: y' = a\cdot y + d
  • x-intercepts: solve a\cdot f(b(x + c)) + d = 0 (do not just map x-intercepts unless d = 0)
  • y-intercept: set x = 0 → y = a\cdot f(b\cdot c) + d
  • Vertical asymptotes x = L map to x = \frac{L}{b} - c
  • Horizontal asymptotes y = L map to y = a\cdot L + d

Absolute Value, Reciprocal, and Inverse

  • y = |f(x)|: Points with y ≥ 0 stay the same; points with y < 0 reflect across the x-axis to (x, −y). Domain unchanged; range becomes [0, ∞) if parent is symmetric about x-axis.
  • y = f(|x|): Keep the right half (x ≥ 0) of y = f(x); mirror it to the left. Domain becomes (−∞, ∞) if f is defined for x ≥ 0; range = range of f on x ≥ 0.
  • y = 1 / f(x): Map (x, y) to (x, 1/y); zeros of f become vertical asymptotes; vertical asymptotes of f remain vertical asymptotes. Where |y| is large, 1/y is near 0; where y is near 0, 1/y is large in magnitude. Domain excludes x where f(x) = 0; range excludes 0 if f has no infinities.
  • y = f⁻¹(x) (inverse): Reflect across y = x: (x, y) → (y, x). Domain and range swap; function must be one-to-one (restrict domain if needed).

Trigonometric Transformations (sin, cos, tan)

  • Standard forms:
    • y = a \cdot \sin(b(x + c)) + d
    • y = a \cdot \cos(b(x + c)) + d
    • y = a \cdot \tan(b(x + c)) + d
  • Common features (sin/cos):
    • Amplitude = |a|
    • Midline = y = d
    • Period (sin/cos) = \frac{2\pi}{|b|} (or 360° / |b| in degrees)
    • Phase shift = −c (left if c > 0, right if c < 0)
    • Range = [ d − |a| , d + |a| ] ; domain = all real x
  • Zeros (anchors):
    • sin: b(x + c) = n\pi \rightarrow x = -c + \frac{n\pi}{b}
    • cos: b(x + c) = \frac{\pi}{2} + n\pi \rightarrow x = -c + \frac{\pi/2 + n\pi}{b}
  • Tangent specifics:
    • Period = \frac{\pi}{|b|}
    • Vertical asymptotes where b(x + c) = \frac{\pi}{2} + n\pi \rightarrow x = -c + \frac{\pi/2 + n\pi}{b}
    • Zeros where b(x + c) = n\pi
    • Domain: all real x except VA lines; range: all real y

Strategy for Sketching/Analysis (Exam Checklist)

  • Rewrite into master form y = a \cdot f(b(x + c)) + d
  • Extract a, b, c, d and record: horizontal scale 1/|b|, shifts, reflections, vertical scale |a|
  • Choose key parent points (intercepts, turning points, zeros/asymptotes), then map each using: x' = \frac{x}{b} - c,\quad y' = a\cdot y + d
  • For trig: compute amplitude, period, phase shift, midline; mark one full period with zeros/extrema/asymptotes
  • For absolute value, reciprocal, inverse: note new domain/range, locate asymptotes (if any), and map representative points
  • Compute intercepts explicitly when d ≠ 0 (do not assume they map from the parent)
  • Label asymptotes, midline, and key points on the final graph; use exact values (π fractions, radicals) until the final step

Short Worked Examples

  • Example 1 (trig): y = 2 \cdot \sin(3(x + \frac{\pi}{6})) - 1
    a = 2 → amplitude 2; b = 3 → period 2π/3; c = π/6 → shift left π/6; d = −1 → midline y = −1
    Zeros: 3(x + π/6) = nπ → x = −π/6 + nπ/3
  • Example 2 (algebraic mapping): f(x) = x² has point (1, 1).
    g(x) = −2 · f( 3 (x + 2) ) + 5
    Map (1, 1): x' = 1/3 − 2 = −5/3 ; y' = −2·1 + 5 = 3 → new point (−5/3, 3)
  • Example 3 (reciprocal): If f has a zero at x = 4, then y = 1/f(x) has a vertical asymptote at x = 4. If f → ±∞ as x → −2, then 1/f(x) → 0, giving a horizontal behavior near y = 0.

Common Pitfalls & Quick Tips

  • Inside vs outside: b and c act horizontally (opposite sense), a and d act vertically (direct sense)
  • “+c” inside means shift left by c; “−c” inside means shift right by c
  • Horizontal scale factor is 1/|b| (not |b|)
  • Negative a reflects across x-axis; negative b reflects across y-axis
  • Intercepts change when d ≠ 0 — recompute them
  • For inverses, confirm one-to-one or restrict domain before reflecting across y = x
  • For tangent, always state period and list VA lines explicitly
  • Always state domain, range, asymptotes, and key points when presenting a final graph
